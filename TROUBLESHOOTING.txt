═══════════════════════════════════════════════════════════════
                    COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════

ISSUE: "Nothing happens after clicking Analyze"
FIX: Wait longer! It can take 10-30 seconds

ISSUE: "Error message appears"
FIX: 
  - Check internet connection
  - Make sure image is not too large (< 5MB)
  - Try a different image
  - Make sure Gemini is selected (not OpenAI)

ISSUE: "App freezes"
FIX: 
  - Close and restart the app
  - Use smaller image
  - Check internet connection

ISSUE: "Can't see the window"
FIX:
  - Check taskbar
  - Press Alt+Tab to find it
  - It might be behind other windows
