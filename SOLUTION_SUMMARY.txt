=============================================================================
                            HOW TO USE
=============================================================================

METHOD 1: GUI (Recommended)
----------------------------
python image_analyzer_ui.py

1. Click "Select Image File"
2. Browse your computer for any image
3. Choose Gemini or OpenAI
4. Click "Analyze Image"
5. See detailed analysis

METHOD 2: Interactive CLI
--------------------------
python image_analyzer_cli.py

1. Enter path to your image file
2. Choose API (1 or 2)
3. Get analysis
4. Analyze another if desired

METHOD 3: Command-Line
----------------------
python image_summarizer.py --image "path/to/image.jpg" --api gemini
